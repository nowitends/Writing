# Quantum Field Theory

## Project status

This file is the local planning document for the formal LaTeX project:

```text
physics_lectures/quantum_field_theory/
```

The course is intended to be complementary to `classical_field_theory_I` and
`classical_field_theory_II`. It should not repeat the classical formalism unless
it is needed to explain how a classical object is converted into a quantum one.

## Working principle

The course should be built in small increments:

```text
small chapter section -> compilation -> PDF review -> next section
```

The visible teaching material should be written in English.

## Core path

The minimal coherent path through the notes is:

```text
canonical quantization
-> Fock space
-> propagators
-> Wick theorem
-> Feynman diagrams
-> scattering
-> loops
-> renormalization
-> QED
-> non-Abelian gauge theory
```
