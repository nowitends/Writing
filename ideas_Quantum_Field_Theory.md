# Recommended Core Path Through the Notes

For a one-semester QFT course complementary to Classical Field Theory I–II, the most coherent path is:

$$
\text{canonical quantization}
\rightarrow
\text{Fock space}
\rightarrow
\text{propagators}
\rightarrow
\text{Wick theorem}
\rightarrow
\text{Feynman diagrams}
\rightarrow
\text{scattering}
\rightarrow
\text{loops}
\rightarrow
\text{renormalization}
\rightarrow
\text{QED}
\rightarrow
\text{non-Abelian gauge theory}.
$$

The minimal core chapters are:

**1–9, 11–14, 16–20, 22–24.**

The extended theoretical path adds:

**25–27, 31–33.**

The phenomenological particle-physics path adds:

**28–30.**
